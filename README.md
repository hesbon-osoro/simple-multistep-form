# Simple MultiStep Form

## Technologies

- React
- TypeScript

## Setup

```code
# Clone this repo
git clone  https://github.com/hesbon-osoro/simple-multistep-form.git

# Navigate to directory and install packages
cd  simple-multistep-form && yarn

# Start the app
yarn start
```
